WAVE LOOM INTERNSHIP PORTFOLIO

Open index.html in any modern web browser.

Files:
- index.html — website content and structure
- styles.css — responsive design and animations
- script.js — navigation, counters, reveal effects, and contact form

The contact form opens the visitor's default email app and drafts a message to waveloom@gmail.com. Google Fonts and Font Awesome icons load from their public CDNs; the site remains readable if those services are unavailable.
