# Waveloom Portfolio

## Overview

Waveloom Portfolio is a responsive portfolio website built using HTML, CSS, and JavaScript. The project showcases a modern web design while demonstrating frontend development practices and cloud deployment workflows.

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Node.js
* Git & GitHub
* GitHub Actions (CI/CD)
* Azure Static Web Apps

## Features

* Responsive user interface
* Interactive frontend functionality
* Automated deployment pipeline using GitHub Actions
* Cloud hosting through Azure Static Web Apps

## Project Structure

```
Waveloom_portfolio/
├─ .github/workflows/
├─ index.html
├─ index.js
├─ script.js
├─ styles.css
├─ package.json
└─ package-lock.json
```

## Deployment

The project is hosted on Azure Static Web Apps. Continuous Integration and Continuous Deployment (CI/CD) are implemented using GitHub Actions, allowing automatic deployment whenever changes are pushed to the repository.
